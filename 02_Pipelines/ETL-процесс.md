
Процесс, в котором очередность действий:
# Extract -> Transform -> Load

Суть: берем данные из источника (s3, реляционные / не реляционные БД), обрабатываем (например, с помощью Python-кода), загружаем очищенные данные в место назначения.

Существует альтернативный [[ELT-процесс]], в котором загрузка и трансформация идут в другом порядке.
